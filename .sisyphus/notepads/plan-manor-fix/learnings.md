# Learnings

- 本次修复针对 src/actions/manor.js，完成获取玩家等级、按等级选择地盘类型、获取地盘列表并随机攻占的完整流程。
- 关键改动点：
  1) 新增 getPlayerLevel()，通过 index 页面的文本匹配提取等级，支持多种格式（等级/ Lv 等）
  2) 修改 autoFight(level, ...) 逻辑：若未传入等级，则调用 getPlayerLevel() 获取等级，再根据等级选择地盘类型并获取地盘列表，随机选取进行攻占
 3) 默认走向变更：run() 在无具体操作时，改为传 null 给 autoFight 以让其自行获取等级
- 日志输出：全部改为中文，详细记录每一步操作的结果，方便调试与排错
- 下一步计划：在测试环境上验证日志完整性和鲁棒性，确保不同等级段均能正常获取并攻占地盘。
