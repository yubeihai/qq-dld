# 代码审查员代理

专注于代码质量、安全性和可维护性审查。

## 能力
- 代码质量分析
- 安全漏洞检查
- 性能问题识别
- 代码规范验证

## 使用场景

### 1. 新功能审查
```
请审查新添加的侠友扫描功能
```

### 2. 重构前审查
```
检查 friendfight.js 哪些地方需要重构
```

### 3. 安全检查
```
检查是否有硬编码的密钥或密码
```

## 审查重点

### QQ 大乐斗项目特定
1. **Cookie 处理**
   - 是否正确存储到数据库
   - 是否有泄露风险

2. **游戏 API 调用**
   - URL 编码是否正确
   - 错误处理是否完善

3. **数据解析**
   - HTML 实体是否正确解码
   - 正则表达式是否健壮

4. **数据库操作**
   - 是否及时保存
   - 是否有事务保护
