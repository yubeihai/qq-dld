# /code-review - 代码审查

审查代码质量、安全性和可维护性。

## 使用方法

```
/code-review "检查 src/game/friendfight.js"
/code-review "审查最近的提交"
```

## 审查清单

### 1. 代码质量
- [ ] 函数职责单一
- [ ] 命名清晰易懂
- [ ] 代码重复度低
- [ ] 错误处理完善

### 2. 安全性
- [ ] 无硬编码密钥
- [ ] 输入已验证
- [ ] 防止注入攻击
- [ ] Cookie 处理安全

### 3. 性能
- [ ] 无多余循环
- [ ] 数据库查询优化
- [ ] 网络请求复用
- [ ] 内存泄漏检查

### 4. 测试
- [ ] 有单元测试
- [ ] 覆盖边界情况
- [ ] 测试可独立运行

### 5. 规范
- [ ] 符合代码风格
- [ ] 注释清晰
- [ ] 提交信息规范

## 输出格式

```markdown
## 代码审查报告

### 文件：[文件路径]

#### ✅ 优点
- ...

#### ⚠️ 需要改进
1. **问题**：描述
   **建议**：修改方案
   **位置**：行号

#### ❌ 严重问题
- ...

### 总体评分：X/10
```
