# /test - 运行测试

运行项目测试并报告结果。

## 使用方法

```
/test                    # 运行所有测试
/test test-friend.js     # 运行指定测试
/test --coverage         # 运行测试并检查覆盖率
```

## 执行步骤

1. **运行测试**
   ```bash
   node test.js
   node test-friend.js
   ```

2. **检查结果**
   - 通过的测试数
   - 失败的测试数
   - 错误信息

3. **覆盖率检查**
   - 核心模块 >= 80%
   - 新功能 100% 覆盖

4. **修复失败测试**
   - 分析失败原因
   - 修复代码或测试
   - 重新运行

## 输出格式

```markdown
## 测试结果

### 总览
- ✅ 通过：X
- ❌ 失败：Y
- ⏭️ 跳过：Z

### 失败详情
1. **测试名称**
   - 错误：...
   - 位置：...
   - 建议修复：...

### 覆盖率
- 整体：XX%
- 核心模块：XX%
```
